NitroType Cheat - Usage

In this guide you will learn how to use the NitroType Cheat

[Return To Home] Cheat is only Available on Windows Devices

Step 1

If you have the cheat installed the usage is fairly straightforward.

You can change the cheats speed with the speed slider and accuracy with the accuracy slider.

If you check the auto start cheat feature then you will need to enter a new race before it works.

God mode will remove the speed slider option entirely and go as fast as possible. It will also run at the max accuracy.
